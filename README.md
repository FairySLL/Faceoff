# Faceoff
Basic turn based combat sequence coded with visual c++ and rendered using SDL 3.

#Faceoff.h
Header file consisting of constants, enums, structs, and classes.

#main.cpp
Main game logic driven by a while loop and a switch statement for the gameState. Utilizes classes to make game logic readable and modular.

#functions.cpp
Function definitions with parameters as defined in Faceoff.h.
